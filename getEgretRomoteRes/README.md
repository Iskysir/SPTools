# getEgretRomoteRes
一键下载获取egret游戏资源包。

## 步骤

1. 安装依赖包： $ npm install

2. 修改配置文件：default.res.json

3. 修改配置路径


```
/*------------------------------------ */
/*-------- 自定义配置区域 ------------ */
/*------------------------------------ */

//-- 资源配置表名称(当前目录下)
var resFileName = "default.res.json";   
//-- 资源根目录路径
var resUrl = "http://xxx.com/resource/";  
//-- 下载资源后的保存路径
var resPath = "./resource/";

/*------------------------------------ */
/*------------------------------------ */
```

4. 执行下载： $ node build.js
